## Overview

This website is a personal portfolio that showcases my skills, tools, projects, and social profiles. It features:  

- Pixel-themed design with **custom fonts**  
- Animated loading screen 🎮  
- Typing effect for name and role  
- Scroll reveal animations for sections  
- Custom pixel cursor  
- Responsive design for desktop, tablet, and mobile  

---

## Features

- **Pixel fonts**: `Pixel-Miners.otf`, `PressStart2P.ttf`, `dpcomic.ttf`  
- **Loading screen**: Animated “LOADING” screen with progress bar  
- **Scroll reveal**: Sections appear with smooth animations  
- **Tools & Projects**: Interactive boxes with pixel borders and hover effects  
- **Socials**: GitHub, LinkedIn, Gmail, Instagram icons with hover animations  
- **Responsive**: Works well on desktop, tablet, and mobile  

---

## Technologies

- HTML5 & CSS3  
- JavaScript (for typing effect & scroll reveal)  
- Firebase Hosting for deployment  
- Custom fonts & pixel art images

## Setup & Deployment

### Prerequisites

- Node.js (v16+ recommended)  
- Firebase CLI installed globally:  
```bash
npm install -g firebase-tools
Local Development

Clone the repository:

git clone https://github.com/ak47kavan/ak.dev.git
cd ak.dev


Serve locally:

firebase serve


or

npx serve .


Open http://localhost:5000 in your browser.

Deploy to Firebase Hosting

Log in to Firebase:

firebase login


Initialize Firebase (if not already):

firebase init hosting


Set public folder to .

Configure as a single-page app? Yes

Deploy:

firebase deploy --only hosting


Your live site will be available at:
https://ak-dev-34ce1.web.app
